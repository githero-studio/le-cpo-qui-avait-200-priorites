# Le CPO qui avait 200 priorités

Tout est priorité 1. Tout est urgent. Tout est stratégique. Tu viens d'être nommée CPO d'une scale-up de 120 personnes qui n'a jamais eu de vraie stratégie produit. Ta roadmap héritée est un Google Sheet de 47 onglets que personne ne consulte. Le backlog fait 1 400 tickets. Le CEO veut "de l'IA". Le CTO veut "réécrire la stack". Les commerciaux veulent "juste ce petit truc pour closer le deal".

Ta mission : transformer ce chaos en vision produit. Spoiler : le chaos gagne. Du moins au début.
